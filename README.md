# THE MD CODE — Corporate Website

## Project Overview
Dual-language (Arabic & English) corporate website for THE MD CODE, featuring:
- The MD Code consulting methodology
- CORE System analytics platform
- Assessment tools and operational resources
- Founder profile and contact system

## Tech Stack
- **Frontend:** HTML5, Tailwind CSS (CDN), Vanilla JavaScript
- **Backend:** Supabase (PostgreSQL, Auth, RLS)
- **Deployment:** Cloudflare Pages via GitHub

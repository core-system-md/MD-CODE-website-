THE MD CODE - Corporate Website

Project Overview:
Dual-language (Arabic & English) corporate website for THE MD CODE.

Tech Stack:
- Frontend: HTML5, Tailwind CSS (CDN), Vanilla JavaScript
- Backend: Supabase (PostgreSQL, Auth, RLS)
- Deployment: Cloudflare Pages via GitHub

Files:
index.html
organization/md-code.html
organization/core-system.html
tools/assessments.html
tools/resources.html
corporate/about.html
contact.html
css/main.css
js/supabase-config.js
js/main.js
assets/logo-gold.png
assets/logo-teal.png
assets/icon-gold.png
assets/founder.png
supabase-schema.sql

Setup:
1. Supabase: Run supabase-schema.sql in SQL Editor
2. GitHub: Upload all files to repository
3. Cloudflare Pages: Connect to GitHub repo, set output to /

Features:
- RTL/LTR language switching
- Forms synced to Supabase
- Protected resource downloads
- Print-friendly layouts
- Mobile responsive
